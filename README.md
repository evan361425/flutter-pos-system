開源的 Flutter 應用程式，希望和使用者一起建立一個好用的 POS 系統（點餐系統）。

## 圖示

![Introduction](https://evan361425.github.io/flutter-pos-system/images/index-introduction.png)

本 POS 系統的特色。

- 完全允許離線使用
- 本系統不會遠端紀錄個資，只會存在你的手機裡，所以可以安心使用
- 庫存系統幫助你紀錄現有成份庫存
- 設定顧客資訊
- 收銀機方便做每日結餘
- 訂單、菜單等資訊的匯出與備份
- 客製化折線圖、圓餅圖的分析
